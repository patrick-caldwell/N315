Terminal Reference

Patrick Caldwell

Application Management

Update all repositories' package lists

sudo apt-get update

Upgrade all installed packages

sudo apt-get upgrade

Install a package

sudo apt-get install [package name]

Remove unused .deb files

sudo apt-get autoclean

Uninstall a package, but leave any config files

sudo apt-get remove [package name]

Uninstall a package

sudo apt-get purge [package name]

Add a repository

sudo add-apt-repository [repository URL]

Web Development

Start the apache server

sudo systemctl start apache2

Stop the apache server

sudo systemctl stop apache2

Restart the apache server

sudo systemctl restart apache2

Start the HTTP server

live-server

Create a package.json file

sudo npm init

Install node-sass

sudo npm install node-sass --save-dev

Run a node.js script

sudo npm run [script]

Connect to IU's VPN

sudo openconnect -b --cafile /etc/ssl/certs/ca-certificates.crt --juniper https://vpn.iu.edu

Disconnect from IU's VPN

sudo killall -SIGINT openconnect
